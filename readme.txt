== Description ==

Embed the Scout lead form into posts/pages via [scout_form] shortcode

== Installation ==

1. From your WordPress Dashboard go to 'Plugins -> Add New' page
2. Click 'Upload Plugin' button
3. Select scout-for-wp.zip archive
4. Click 'Activate'
5. Go to 'Settings -> Scout for WP' page and enter your API key and click 'Save Changes' button
